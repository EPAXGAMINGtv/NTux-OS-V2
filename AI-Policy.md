# AI Policy for NTux-OS

AI tools and agents (opencode, GitHub Copilot, Cursor, ChatGPT, etc.) are
generally allowed for all parts of this project — kernel, drivers,
userspace, documentation, and anything else.

## Rules

1. **Pull requests must be labeled** if they were created or significantly
   assisted by AI. Add a comment or use a `[ai]` tag in the title so
   reviewers know.

2. **The committer is always responsible.** You must read, understand, and
   manually review every line before committing. Do not commit code you
   cannot explain.

3. **AI may not be listed as author or co-author.** Only humans can be
   authors.
